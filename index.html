MIA DB

is Multi-user (and Multi-tenant) Interactive Analytical database developed by Briskat. Revolutionary architecture combines concepts from the outside world ( e.g. Git, Dropbox, Erlang ) with our previous experience in OLAP industry.

MIA enables fast development of various kind of analytics. At the same time, it delivers great online user experience, reduces complexity and the total cost of an interactive online analytic solution. Million daily users can easily analyze data while getting near-instant responses.

Key design features:

  • high requests throughput
  • low latency responses
  • ad-hoc querying
  • data agnostic
  • frequent data loads
  • frequent changes in a data model
