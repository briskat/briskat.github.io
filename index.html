MIA DB

is Multi-user and Multi-tenant Interactive Analytical database developed by Briskat.

MIA DB is optimized for high requests throughput and low latency responses. Millions of daily users can analyze a lot of data while getting near-instant responses.

Our revolutionary architecture combines concepts from outside world ( e.g. Git, Dropbox, Erlang ) with our previous experience in OLAP industry.

Not only it allows to avoid costly manual tunning of queries. It's also designed for frequent data loads, frequent changes in a data model and ad-hoc querying.

It enables fast development of various kind of analytics at the same time with scalability and great user experience.
